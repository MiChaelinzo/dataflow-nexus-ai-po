# Quick Start Guide

Get up and running with the Analytics Intelligence Platform in minutes.

## üöÄ Getting Started

### 1. Launch the Application
The application is pre-configured and automatically starts in your Spark Codespace. Simply open the preview URL to access the platform.

### 2. First-Time Setup
On your first visit, you'll see a welcome screen introducing the platform's capabilities. Click **"Get Started"** to proceed to the main dashboard.

### 3. Authentication
Click **"Sign in with GitHub"** to authenticate. This enables:
- Personalized user profile
- Activity tracking
- Workspace management
- Team collaboration features

## üìä Exploring Features

### View the Dashboard
The main dashboard shows:
- **Key Metrics**: Revenue, users, conversion rate, engagement score
- **Trend Charts**: 30-day revenue history
- **Segment Breakdown**: Revenue by category
- **Feature Cards**: Quick access to advanced capabilities

### Generate AI Insights
1. Navigate to the **"AI Insights"** tab
2. Click **"Generate Insights"** button
3. AI analyzes current metrics and surfaces actionable findings
4. Each insight shows:
   - Priority level (Critical, High, Medium, Low)
   - Confidence score (60-95%)
   - Specific recommendations
5. Bookmark important insights for later reference

### Explore Tableau Pulse
1. Go to the **"Pulse"** tab
2. View automatically generated insights
3. Filter by:
   - Priority level
   - Insight type (Anomaly, Trend, Forecast, etc.)
   - Read/unread status
   - Bookmarked items
4. Click **"Notifications"** to configure Slack integration
5. Set up automated alert rules

### Try Seasonal Analysis
1. Navigate to **"Seasonal"** tab
2. View automatically detected seasonal patterns
3. Review:
   - Seasonality strength (Strong/Moderate/Weak)
   - Peak and low periods
   - Quarterly forecasts
4. Check priority recommendations
5. Mark recommendations as "Applied" when implemented

### View Predictions
1. Open the **"Predictions"** tab
2. See 14-day revenue forecast with confidence intervals
3. Review prediction metrics:
   - Predicted growth percentage
   - Confidence level
   - Expected revenue

### Use Natural Language Queries
1. Go to **"Semantic"** tab
2. Browse the data catalog
3. Click on any metric to see:
   - Business definition
   - Calculation logic
   - Related metrics
4. Use the search to find specific data points

## üë• Collaboration

### See Team Activity
1. Navigate to **"Collaborate"** tab
2. View live presence indicators showing who's online
3. See real-time cursor positions of other users
4. Watch the activity feed for team actions

### Create a Workspace
1. Go to **"Workspaces"** tab
2. Click **"Create Workspace"**
3. Enter workspace name and description
4. Add team members
5. Assign roles (Owner, Editor, Viewer)

### Share a Dashboard
1. Navigate to **"Shared"** tab
2. Click **"Share New Dashboard"**
3. Enter dashboard details
4. Set permission level:
   - Public: Anyone with link
   - Team: Workspace members only
   - Private: Specific users only
5. Copy share link

### Record a Session
1. Go to **"Replay"** tab
2. Click **"Start Recording"**
3. Navigate through tabs and interact with features
4. Click **"Stop Recording"**
5. Access saved recordings from the list
6. Click **"Play"** to watch with full playback controls

## üìà Reports & Analysis

### Build a Custom Report
1. Navigate to **"Reports"** tab
2. Select report components:
   - Key metrics
   - Time series charts
   - Top insights
   - Category breakdowns
3. Configure date range
4. Click **"Generate Report"**
5. Preview in browser or download as PDF

### Schedule Automated Reports
1. In **"Reports"** tab, go to **"Scheduled Reports"**
2. Click **"Schedule New Report"**
3. Configure:
   - Report name
   - Frequency (Daily, Weekly, Monthly)
   - Recipients (email addresses)
   - Included sections
4. Enable the schedule
5. Reports deliver automatically via email

### Compare Time Periods
1. Go to **"Compare"** tab
2. Select **"Period Comparison"**
3. Choose date ranges to compare
4. Select metrics to analyze
5. Review variance analysis
6. Export comparison report

### Year-over-Year Analysis
1. In **"Compare"** tab, select **"Year-over-Year"**
2. View monthly comparison bars
3. Review seasonal trend analysis
4. Identify best/worst performing months
5. Check quarterly summaries

## üÜò Troubleshooting

### Insights Not Generating
- Check your internet connection
- Ensure LLM service is available
- Try refreshing the page
- Check browser console for errors

### Collaboration Features Not Working
- Verify you're signed in
- Check that WebSocket connection is active
- Ensure other users are on the same workspace
- Try clearing browser cache

### Tableau Content Not Loading
- Verify Tableau Server URL
- Check authentication credentials
- Ensure embed permissions are set
- Review browser console for CORS errors

### Slack Notifications Not Sending
- Verify Slack connection is active
- Check notification rules are enabled
- Ensure selected channel exists
- Test with manual notification first
